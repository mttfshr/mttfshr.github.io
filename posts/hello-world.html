Hello again world

This is a place to muck around with a few ideas about what a personal website is for & what tools exist to DIY as much as possible. Everything is an experiment.

A couple ideas/goals/motivators:

  • Social networks by and large give me the creeps, for reasons I may write down eventually
  • It's looking pretty clear that the post-social network landscape has us owning and controlling the locations of our own content, which in a way is more like the old days and is something I'm looking forward/backward to
  • I want a central place to not just post things but to reflect on the stuff I've posted
  • I want to control the presentation too, and want flexibility to organize and show things maybe in ways that aren't the regular way
  • I want something that functions more like a note/sketchbook than a portfolio (I have a portfolio site already)

So, here's the general plan/idea:

  • Play around with POSSE/ActivityPub/Social Linked Data/P2P (also this) and see what might work
  • Make things and put them here. Have per thing options for a social layer/syndication/sharing elsewhere to put them in front of eyeballs, but they live here.
  • Control & fuck with page templating & layout ideas
  • See if there's a way to hybridize local file & note creation/storage with a web publishing workflow vis a vis static site generation
  • As much as possible, utilize a note taking & sketching workflow that doesn't rely on proprietary formats or software & is portable

The recipe right now:

  • This site is written in Markdown text files and compiled using Saber then pushed to a Github repo to be served via Github pages
  • Media storage (esp. streaming files) is still a question mark

Let's see where this leads
